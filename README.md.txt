# Automated File Organizer (Python)

A Python automation script that organizes files in a directory into folders based on their file types.  
It automatically creates folders, moves files, and logs every action with timestamps.

---

## Features
- Automatically categorizes files by extension
- Creates folders if they do not exist
- Organizes files into Images, Documents, Videos, Music, etc.
- Handles unknown file types by placing them in an "Others" folder
- Logs all file movements with date and time

---

## Technologies Used
- Python
- os module
- shutil module
- datetime module

## How to Run
1. Clone or download the repository
2. Create a folder named `test_folder` in the project directory
3. Add random files (pdf, jpg, mp3, etc.) inside `test_folder`
4. Run the script:
5. Files will be organized automatically

---

## Sample Use Case
This project is useful for organizing messy folders such as Downloads, Desktop files, or project directories automatically.
